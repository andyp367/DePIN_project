DePIN Network Uptime Reward Project
Project Overview

This project is a decentralized physical infrastructure network (DePIN) system designed to track network uptime and reward users the longer they remain active on the network. A physical device monitors uptime data and reports activity to the system. Based on verified uptime, users earn token rewards.

The project combines embedded hardware, blockchain concepts, validators, wallets, and a live dashboard interface. Validators confirm uptime submissions and add approved transactions to the blockchain, while wallets store earned rewards. The dashboard provides real-time visibility into balances, validator activity, and newly added blocks.

Core Features
- Track user/device network uptime
- Reward users based on verified active time
- Blockchain ledger for transaction records
- Validator nodes to approve submissions
- Wallet system for token balances
- Live dashboard for network monitoring

System Flow
- Device stays active on the network and tracks uptime.
- Uptime data is submitted to the system.
- Validators verify the submission.
- A new blockchain entry is created.
- Rewards are added to the user wallet.
- Dashboard updates in real time.
